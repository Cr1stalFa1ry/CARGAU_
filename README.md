Этот проект является продолжением моего предыдущего приложения — [cargau](https://github.com/Cr1stalFa1ry/cargau).
Почему решил сделать новый? Да просто стало интересно поработать с CQRS + mediatr.

Проект выполнен на ASP.NET Core, использует Entity Framework Core, PostgreSQL, JWT + Refresh Token, а также принципы Clean Architecture.
Основная цель — улучшить структуру кода, добавить в дальнейшем кеширование, логгирование, обработку ошибок в виде ответа пользователю от сервера, упростить масштабирование и повысить тестируемость благодаря CQRS-подходу.
